  Press q during inventory screen to quit
  commands:
  equip <item> : Used on the inventory screen to equip or use an item
  whereami : Display current location
  whoami : Display your name
  move <Location> : Move to selected Location
  save : Save the game
  quit : Quit the game, note you will not be prompted to save before!
  reload : Reload your last save game
  inventory : Open up inventory screen to see or use items
  explore <POI> : Explore the selected POI
